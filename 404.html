Get Assured

Summary:

GetAssured is Singapore’s first life insurance based employee benefits portal. It provides a platform for companies to help their employees look for the best value life insurance products.

• Implement multi-tenancy functionality to the GetAssured platform to create a unique site experience for each individual business clients. • Design and Convert the platform’s front end into React.js framework in order to facilitate future improvements as well as enhance customer’s experience. • Conduct bug fixes and enhancement to improve the general stability and usability of the platform. • Isolate crawler bot and convert it into its own micro services with api for the main server so as to relieve stress on main server. • Set up, deploy and managed application using AWS web services such as EC2, S3.

Technologies Used:

  • React
  • Ruby On Rails
