Summary:

They get an inventory list and expiry date for each item on their phone as well as receive reminders before the food goes bad.

All Assured

Summary:

During my time in TrunkTech, the second project that I was involved in was the app AllAssured. It is a robo-assisted insurance optimiser that allows individuals to plan and optimise their spending on their insurance portfolio.

In the preliminary phase, I helped to manage the product development of the application all the way from prototype to launch, ensuring code developed meets the requirements of management with an outsource team.

As it is always better to have an in house developer, I took it upon myself to pick up an unfamiliar language and framework, PHP + Laravel and helped to refactor, and add in enhancement in the backend to ensure the application meets the company’s business needs.

Also I proposed, implemented a redundancy system (scheduled database backups, snapshots, seeder files) to ensure that we can recover in events of data losses.

As the original website was built using Laravel frontend and basic jquery, I planned and integrated React with Laravel backend and built a web front end for our mobile apps.

Technologies Used:

  • Laravel
  • MySql
  • AWS EC2, RDS
  • React
